## クラス基本設計書
  クラスの基本設計書になります。


### 概要
  そのクラスがどんな処理を行うのかを記述します。

### クラスの役割

  外部に対して何を行っているのかをテーブル形式で記述します。
  - 役割
  - メソッド名
  - 引数
  - 戻り値
  - 例外処理

### 依存関係
  どのクラスを参照して、どのクラスから参照されるのかを記述します。
  継承などある場合は、そちらも記述します。

  参照している場合は、下記を記述します。
  - どのクラスのどのメソッド（変数）を、どのメソッドで使用しているか

  参照されている場合は、下記を記述します。
  - どのクラスのどのメソッドから、どのメソッド（変数）が呼ばれているか
